new CheckboxModal(init)

Bootstrap modal which displays tables containing checkboxes or radio buttons, which can be toggled individually or through a global checkbox

Parameters:
Name Type Description
init CheckboxModalOptions
Source:
  • checkbox-modal.js, line 17
Example
new CheckboxModal({
    i18n: i18n,
    vent: vent,
    parameters: {
        modalHeader: 'modalHeader'
    },
    tables: [{
        initialState: true,
        tableHeader: 'tableTitle',
        rows: [
            'one',
            'two',
            'three'
        ]
    }]
});
